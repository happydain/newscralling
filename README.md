# AI 뉴스 요약 및 CSV 추출 도구

이 앱은 Google Gemini AI를 사용하여 최신 뉴스를 검색하고 요약해줍니다.

## 🛠 실행 방법

1. GitHub Codespaces를 엽니다.
2. 터미널에 아래 명령어를 입력합니다:

```bash
pip install -r requirements.txt
streamlit run app.py
```

3. Streamlit 화면이 열리면 사이드바에 Gemini API Key를 입력합니다.
4. 검색 키워드를 입력한 뒤 뉴스 검색 및 요약을 시작합니다.

## 📁 파일 구성

- `app.py`: Streamlit 앱 실행 파일
- `requirements.txt`: 필요한 Python 패키지 목록
- `README.md`: 실행 방법 안내 문서
